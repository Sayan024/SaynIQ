# 🧠 SaynIQ - Premium Landing Page

This branch contains the source code for the **SaynIQ Official Website**.

## 🚀 Branch Structure

- **`main`**: SaynIQ Website (Current Branch)
- **`sayniq-app`**: SaynIQ Mobile App (React Native / Expo)

---

## 💻 Website Setup

The landing page is built with **React**, **Tailwind CSS**, and **Framer Motion**.

### Setup Steps:
1. Clone the repository and ensure you are on the `main` branch.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

---

## 📱 Switching to Mobile App

To access the mobile app source code, switch to the `sayniq-app` branch:

```bash
git checkout sayniq-app
```
